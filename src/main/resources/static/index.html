    关于我们

    山东嘉鹏木业有限公司，成立于1999年，占地20多公顷，位于全国闻名的板材生产基地-山东省临沂市，工厂西临京沪高速公路,东接307国道,交通位置十分优越.拥有超过1500多名员工。隶属于山东鹏程集团，山东鹏程集团旗下拥有5家子公司，其中包括木业公司、长条地毯公司、骰子板公司，木业机械公司和山东鹏程国际贸易有限公司。山东嘉鹏木业有限公司主要生产地毯钉条、高档实木墙板、实木工艺品、实木指接板。
